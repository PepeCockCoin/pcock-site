
# Pepe Cock | $PCOCK — Meme Site

**How to deploy on GitHub Pages**

1. Upload everything in this folder to your repo root (`pcock-site`).
2. Repo → Settings → Pages → Source: *Deploy from a branch* → Branch: `main` → Folder: `/ (root)` → Save.
3. Open https://YOUR-USER.github.io/pcock-site

**Swap Telegram later**
- Find the two buttons with `Join Telegram (soon)` and replace `class="btn warn disabled"` with `class="btn"`,
- set the `href="https://t.me/YOURGROUP"` and edit the label.

**Change Buy link after graduation**
- Replace the pump.fun URL in the two buttons with your Raydium/Jupiter link.
